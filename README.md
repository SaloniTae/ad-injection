# Ad Injection Platform

This repository contains the ad-injection platform with a fixed-role player, glassmorphism admin UI, iframe embed control, and auto-detect link format.

## Features
- Image and video ad timing fixes
- Resume behavior improvements
- Iframe embed control via IFRAME_ALLOW_ORIGINS
- Glassmorphism SaaS UI for admin panel
- Auto-detect URL type for ads

## Getting Started
1. Install dependencies: `npm install` in the `server` directory.
2. Set environment variables in `.env` (see `.env.example`).
3. Run the server: `node src/index.js`.
